h1. CakePHP Twitter API Plugin

A CakePHP Plugin containing a DataSource for interarcting with Twitter web services.

h2. Dependencies

CakePHP HttpSocket from neilcrookes https://github.com/neilcrookes/http_socket_oauth (Vendor)
CakePHP ReST DataSource Plugin https://github.com/neilcrookes/CakePHP-ReST-DataSource-Plugin

h2. Configuration

Configure and use config/twitter_config.php.default
Clone CakePHP ReST DataSource Plugin under plugins/rest

h2. Usage

bc. 
var $uses = array('Twitter.TwitterStatus');
...
if (($tweets = Cache::read('TwitterStatus.userTimeline')) === false) {
	$tweets = $this->TwitterStatus->find('userTimeline', array('include_rts'=>true));
	Cache::write('TwitterStatus.userTimeline', $tweets);
}
